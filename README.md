#  Basit Hile Satış sitesi gelişitirilmeye acık.

Selamlar CheatGlobal üyeleri,

Sizler için tamamen modern bir UI/UX anlayışıyla, üst düzey animasyonlarla desteklenmiş ve Full-Stack olarak entegre edilebilecek **FiveM Hile Satış & Yönetim Scriptini** tanıtıyorum. Sadece bir tanıtım sayfası değil, arka planda lisans anahtarı (key) üretebileceğiniz, kullanıcıları yönetebileceğiniz entegre bir Admin Paneline de sahiptir.

##  Öne Çıkan Özellikler

* **Modern Landing Page**: Ziyaretçileri satın almaya ikna edecek, karanlık tema (dark mode) üzerine kurulu SaaS tarzı premium görünüm.
* **Gelişmiş Animasyonlar**: Sayfa geçişleri, hover efektleri ve özel imleç (custom cursor) tasarımı.
* **İnteraktif Hile Arayüzü Önizlemesi**: Müşterilere hilenizin nasıl göründüğünü sayfa üzerinden hissettiren UI Mockup.
* **Kapsamlı Admin Paneli**:
  * Kullanıcı yönetimi (Kayıtlı kullanıcıları, rolleri ve kayıt tarihlerini görme)
  * Dinamik Lisans (Key) Üretimi (Haftalık, Aylık, Sınırsız / Lifetime seçenekleriyle)
  * Lisans durum takibi (Kullanımda / Boşta tespiti, kullanan User ID)
  * Key silme / iptal etme özellikleri
* **Auth Sistemi**: JWT tabanlı Login ve Register sayfaları, dinamik yetkilendirme.
* **Duyarlı Tasarım (Responsive)**: Mobil, tablet ve masaüstü tüm cihazlarda kusursuz görünüm.

## 💻 Kullanılan Teknolojiler ve Kütüphaneler

Proje en güncel web teknolojileri kullanılarak inşa edilmiştir:

* **React.js**: Kullanıcı arayüzü ve state yönetimi için (Hooks, Context).
* **Framer Motion**: Sayfadaki tüm akıcı animasyonlar, popup açılışları ve element geçişleri için kullanıldı. (Piyasadaki en iyi React animasyon kütüphanesi).
* **Tailwind CSS**: Hızlı, modern ve estetik stillendirme için. Özel renk paletleri ve shadow efektleri Tailwind ile yapıldı.
* **Lucide React**: Premium ve modern ikon seti. (Crosshair, Eye, ShieldCheck, Zap vb. ikonlar).
* **Node.js & Express (Backend)**: (Scriptin bağlanacağı API altyapısı - JWT, REST API tabanlı).

## 📸 Görüntüler / Tasarım Detayları

* **Özel İmleç (Custom Cursor)**: Etkileşimli öğelerin üzerine gelindiğinde büyüyen, sayfaya derinlik katan modern imleç.
* **Paket Seçimi (Pricing)**: Kripto ödeme entegrasyonuna hazır, popüler paket vurgulu fiyatlandırma kartları.
* **Özellik Sekmeleri**: Aimbot, Visuals ve Protection gibi hile özelliklerini dinamik sekmelerle gösteren interaktif alan.
